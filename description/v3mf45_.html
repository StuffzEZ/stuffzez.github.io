How To Install Minecraft Mods


1) Choose Your Loader

If You Want To Use Server Side Mods Like Create Then Choose Forge.

If You Want To Use Client Side Mods Like Sodium That You Can Use On Any Server Then Choose Fabric.

If You Have Got A Low-End Computer Then I Recommend Fabric Becuase If You Install Some Performance Mods, It Improves How Minecraft Performs And You Get A Massive FPS Boost.


2) Download Your Loader.

To Install Fabric, Go To https://fabricmc.net.

To Install Forge, Go To https://files.minecraftforge.net.


3) Download Your Mods

Personally, I Use https://modinth.com To Download Mods But You Can Use Other Websites.

Once On The Website, Search For The Mod You Want To Download (For Fabric You Have To Have Fabric API For It To Work).

Then, Click On It, (On Modrinth) Go To Verisons, Then Download The Version That Works With The Version Of Your Loader.

Once You Have Downloaded Your Mods, Copy Them To C:/Users/[YOUR USERNAME]/AppData/Roaming/.minecraft/mods (Replace [YOUR USERNAME] With Your Username).


4) Play Minecraft.

Open The Launcher.

Click Installations.

Find Your Loader's Installation And Press The Green Play Button Next To It.
